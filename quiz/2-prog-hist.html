Quiz 2: Programming History

by Amir Shahmoradi — 03 Oct 2022

This quiz aims at testing your basic knowledge of programming history.
Don’t forget to push your answers to your remote repository by the end of quiz time.
Push all your answers to quiz/2/ folder in your Github project.


  1. What is the name of the fastest part of a computer memory?

    Answer: Register

  2. What is the slowest kind of computer storage device?

    Answer: External or Hard drive.

  3. Name the hierarchy of the memory of modern computers, from the smallest to the largest memory size?

    Answer: Register, L1, L2, L3, RAM, External or Hard drive.

  4. What is smallest unit of information in computers? (Hint: Recall our computer programming lecture)

    Answer: bit.

  5. Why humans have chosen the binary numbers (0 and 1) to represent data in computers?

    Answer: Because binary information is readily representable by electric currents.

  6. What is the typical speed of contemporary computer registers?

    Answer: On the order of nanoseconds.

  7. What decade was the first digital computer built?

    Answer: 1940’

  8. What is a fixed program computer?

    Answer: Early models of computers that could only perform certain tasks.

  9. Name an example of modern-day fixed program computer.

    Answer: Calculators.

  10. What is the difference between declarative and imperative knowledge?

    Answer: Declarative knowledge involves statements of fact. Imperative knowledge is about how to accomplish something, or basically, is a recipe on how to do something, known as algorithm in scientific terminology.

  11. What class of human knowledge algorithms belong to?

    Answer: Imperative knowledge.

  12. What is a computer bug?

    Answer: An error in program that has to be resolved for successful execution of the program.

  13. What is a compiler? (Hint: Recall the glossary of our lectures)

    Answer: A software that translates an entire high-level program into a lower-level language to make it executable.

  14. What is the name of the first digital computer?

    Answer: ENIAC.

  15. Name a computer programming language whose evolution and popularity is tightly coupled with the field of Bioinformatics.

    Answer: Python.

  16. Name a computer programming language whose evolution and popularity is tightly coupled with the field of Numerical Computing, Atmospheric Science, Plasma Sciences, and Aerospace Engineering, among others.

    Answer: Fortran.

  17. Name the five major components of a modern digital computer.

    Answer: input device, CPU, main memory, external storage, output device.

  18. Name an example of fixed-program computer.

    Answer: Calculators.

  19. Order the following languages chronologically according to their birth date. (Hint: Use the diagrams from our lecture notes)
    Java, COBOL, C++, Python, Fortran, B

    Answer: Fortran, COBOL, B, C++, Java, Python

  20. (A) What is the closest programming language to machine code (i.e., binary code)?

    Answer: Assembly

    (B) Does it need interpretation in order to become machine-comprehensible?

    Answer: Yes. An Assembler interprets the program for the machine.

  21. (A) Name the oldest high-level programming language that is still in active daily use.

    Answer: Fortran

    (B) Approximately how many decades is it old? ($\pm15$ years is acceptable answer. the decade it was created is also an acceptable answer)

    Answer: in 1950s

  22. (A) Name a second-generation programming language.

    Answer: Assembly

    (B) Which language-generation are Fortran, C, C++, MATLAB, Python, R?

    Answer: third, third, third, fourth, fourth, fourth

  23. In what decades C, C++, and MATLAB/Python were created, respectively? (Hint: Use the diagrams from our lecture notes)

    Answer: 1970s, 1980s, 1980s, 1990s

  24. Name an ancestor programming language of C (Hint: Use the diagrams from our lecture notes).

    Answer: B

  25. Name a programming language ancestor of C++ (Hint: Use the diagrams from our lecture notes).

    Answer: C, Simula

  26. Name a programming language ancestor of MATLAB and a programming language ancestor of Python (Hint: Use the diagrams from our lecture notes).

    Answer: Fortran/C

  27. How do you define logarithm? Explain with a simple example just as we did in class.

    Answer: The logarithm in bae of 10, the exponent that if the number 10 is raised to, then it yields the original number. For example, log10(100) = 2.

  28. Extra Credit. Which memory device is longer lasting: magnetic tapes or HDD (Hard-Disk Drives)? How many years approximately each one last (a rough reasonable estimate is good enough)?

    Answer: Magnetic last decades, HHD last around 5-10 years.
